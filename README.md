## 📊 Project Overview  

The **Banking Analytics Dashboard** is a comprehensive data analytics project built using **Power BI** to analyze financial data across transactions, customer accounts, credit risk, and branch performance.  

This dashboard transforms raw data into **interactive visualizations and actionable insights** to support better decision-making in banking operations.  

---

## 🎯 Business Objectives  

- Analyze customer transaction behavior  
- Monitor financial KPIs and performance  
- Evaluate credit risk and loan stress  
- Track branch-level performance  
- Enable data-driven business decisions  

---

## 📂 Data Sources  

### 1️⃣ Banking Transactions Dataset  
- Transaction Type (Deposit, Withdrawal, Transfer, Payment)  
- Transaction Amount & Currency  
- Date & Time (Morning, Afternoon, Evening, Night)  
- Branch Code  

### 2️⃣ Customer Account Dataset  
- Account Type (Savings, Checking, Loan, Credit)  
- Account Balance  
- Credit Score  
- Loan Amount & Interest Rate

## 📈 Key Insights  

### 💳 Transaction Analysis  
- Withdrawals dominate transaction volume  
- Peak activity during morning & afternoon  
- Weekday transactions higher than weekends  

### 💰 Financial Insights  
- Net Positive Cash Flow: **+$16.8M**  
- Strong transaction volume across all channels  
- Year-over-year decline observed  

### ⚠️ Risk Analysis  
- Avg Credit Score: **692**  
- High-risk accounts: ~1.4%  
- Loan stress concentrated in low credit score segments  

### 🏦 Branch Performance  
- Total Branches: **600**  
- Top Branch: **BR-0532**  
- Majority of revenue from top-performing branches  

### 👥 Customer Insights  
- Savings accounts hold highest balances  
- Technology & Finance sectors dominate  
- Majority customers are long-term (loyal)
